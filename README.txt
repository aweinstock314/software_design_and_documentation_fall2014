GitHub repository for Software Design & Documentation class 
project, for the team consisting of Kyle Kerega, Kate 
McGuire, and Avi Weinstock.

Instructions to run on a new system (requires a PostgreSQL install, ANT, a JDK, and a bourne-compatible shell):
Ensure that the database is set up, and put your postgres username/password pair on seperate lines in "project/epic_database_creds.txt".
Run "setup_and_run_epic_webserver.sh" with a bourne-compatible shell (such as sh or bash).
Connect via your favorite web browser at "localhost:8000".
